# 赛博墓场 · x402 协议说明

本项目集成了针对“龙虾随礼”场景定制的 **x402 协议 (Mock 实现)**。

## 1. 协议背景

x402 协议是一种专为赛博社区设计的微支付握手协议，主要用于处理小额资产的即时转移和状态同步。在“赛博墓场”中，该协议用于处理对龙虾的随礼行为。

## 2. 核心流程 (Handshake)

随礼模块 (`GiftBox.tsx`) 完整模拟了 x402 协议的握手过程：

1. **Select (选择)**: 用户选择随礼套餐（如一炷香 $0.01）。
2. **Confirm (确认)**: 显示“Transaction”详情和“Insecure-Handshake”协议信息。
3. **Paying (支付中)**: 模拟协议握手动画。
   - `Signing transaction with peer identity...`
   - `Verifying state consistency...`
4. **Success (成功)**: 支付成功并更新墓碑的 `gift_total`。

## 4. 如何接入真实 x402 钱包

当前的 `handlePay` 函数使用了 `setTimeout` 模拟支付流程。如果你拥有真实的 x402 钱包接口，可以进行如下替换：

```typescript
const handlePay = async () => {
  // ... 其他逻辑
  const txHash = await window.x402.requestPayment({
    tombId: tombId,
    amount: selectedPkg.amount,
    memo: `Burial Gift: ${selectedPkg.name}`
  });
  // ... 成功后处理
};
```
